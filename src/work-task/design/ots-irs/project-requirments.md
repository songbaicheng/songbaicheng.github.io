# 项目需求
## 背景
目前我行代客利率掉期[^one]交易的询价、报价、审批、交易以及账务处理都是采用手工方式，需要等待总分行之间反复确认，时效性低，并且操作风险性高，为了完善为企业客户提升更全面、方便、快捷的服务，并且在同业具备自动化代客利率掉期交易的功能基础上，开始着手实现自动化的线上代客利率掉期交易。

[^one]: 代客利率掉期，指采取只进行利息轧差计算，不交换名义本金的方式，在未来的一定期限内，根据约定的名义本金和利率进行一次或多次利息交换的产品。
[^two]: 轧差，在某一时点，某项被核算的资金或科目的资金来源和运用差额的统称。
